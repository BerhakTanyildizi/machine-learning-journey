# 🚀 Machine Learning Journey

Welcome to my Machine Learning repository! This project serves as a comprehensive, structured log of my hands-on experience and continuous learning in Data Science and Machine Learning. 

My primary focus in this repository is to demonstrate practical knowledge and end-to-end implementation skills, heavily emphasizing **Data Cleaning, Exploratory Data Analysis (EDA), Feature Engineering**, and the deployment of various **Machine Learning Algorithms**.

## 🛠️ Technologies & Libraries
- **Data Manipulation & Processing:** `pandas`, `NumPy`
- **Data Visualization:** `Matplotlib`, `Seaborn`
- **Machine Learning Frameworks:** `scikit-learn`, `XGBoost`, `LightGBM`

## 📂 Project Structure
To maintain a clean and professional workspace, the notebooks and datasets are organized into the following logical directories:

### `01-Data_Cleaning_and_Preprocessing/`
* Focuses on the crucial first steps of the data pipeline: handling missing values, encoding categorical variables, feature scaling, and exploratory data analysis (EDA).

### `02-Regression_Models/`
* Simple Linear Regression
* Multiple Linear Regression
* Polynomial Regression
* Regularization Models (Ridge, Lasso, ElasticNet)
* Support Vector Regression (SVR)
* K-Nearest Neighbors (KNN) Regressor
* Ensemble Regressors (AdaBoost Regressor, Gradient Boosting Regressor)

### `03-Classification_Models/`
* Logistic Regression (Binary, Multi-Class, and Advanced implementations)
* Support Vector Machines (SVM Classifier)
* Naive Bayes Classifier
* K-Nearest Neighbors (KNN) Classifier
* Decision Tree Classifier
* Ensemble Classifiers (Random Forest, AdaBoost, LightGBM, Gradient Boosting, XGBoost)

### `04-Unsupervised_Learning/`
* **Clustering:** K-Means Clustering, HDBSCAN
* **Dimensionality Reduction:** Principal Component Analysis (PCA)

## 🧠 What's Inside?
Each notebook in this repository typically follows a structured data science workflow:
1. **Data Loading & Inspection:** Understanding the raw data.
2. **Data Cleaning:** Treating null values, outliers, and duplicates.
3. **EDA (Exploratory Data Analysis):** Visualizing relationships and distributions using Seaborn and Matplotlib.
4. **Model Building & Training:** Implementing the appropriate algorithm based on the problem type.
5. **Model Evaluation:** Assessing performance using relevant metrics (Accuracy, F1-Score, RMSE, R-squared, etc.).
